## Project Title
Build an AI Chatbot Using LLMs

## Description
AI Study Assistant is a simple AI-based chatbot application developed using Python.  
The application allows users to ask academic questions and receive answers generated by a Large Language Model (LLM) in different explanation styles.

This project demonstrates the basic working of a full-stack AI application by integrating a frontend, backend, and an AI model API.

---

## Objective
The objective of this project is to design and implement a full-stack AI chatbot application using modern AI tools and software engineering practices.

## System Architecture
1. The user enters a question in the Streamlit frontend.
2. The frontend sends the request to the FastAPI backend.
3. The backend forwards the question to the LLM API.
4. The AI model generates a response.
5. The backend sends the response back to the frontend.
6. The answer is displayed to the user.

This follows a simple requestâ€“response client-server architecture.

## How to Run the Project
### Step 1: Install Dependencies
```bash
pip install fastapi uvicorn streamlit requests
```

### Step 2: Run Backend (FastAPI)
```bash
cd backend
uvicorn main:app --reload
```

### Step 3: Run Frontend (Streamlit)
```bash
cd frontend
streamlit run app.py
```
